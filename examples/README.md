# Examples

```bash
uv run slopmeter
uv run slopmeter --codex --dark
uv run slopmeter export --format html --output ./examples/heatmap.html
uv run slopmeter export --codex --format json --output ./examples/codex.json
uv run slopmeter export --all --dark --format svg --output ./examples/all-dark.svg
```
